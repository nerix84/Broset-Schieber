# Brøset-Schieber BVC-CH

Statische Web-App für GitHub Pages.

## Veröffentlichung über GitHub Pages

1. Neues GitHub-Repository erstellen, z. B. `broset-schieber`.
2. Diese Dateien in das Repository hochladen.
3. In GitHub: Settings → Pages.
4. Source: „Deploy from a branch“.
5. Branch: `main`, Ordner: `/root`, speichern.
6. Nach kurzer Zeit ist die App unter `https://DEINNAME.github.io/broset-schieber/` erreichbar.

## iPhone

URL in Safari öffnen → Teilen-Symbol → „Zum Home-Bildschirm“.
